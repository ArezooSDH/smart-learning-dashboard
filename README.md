# Smart Learning Dashboard

A modern CRUD dashboard for managing students and courses built with Next.js.

## Features

- Student management
- Course management
- Search / sort / pagination
- Responsive dashboard
- Reusable data table
- Form validation with Zod
- React Hook Form
- Toast notifications
- Confirmation dialogs

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- TailwindCSS
- Shadcn UI
- Zod
- React Hook Form
- Axios

## Installation
```bash
npm install
npm run dev
